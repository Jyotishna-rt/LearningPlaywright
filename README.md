# LearningPlaywright

Repository for Playwright learning exercises and example scripts.

## Structure

- Chapter_01_Basics/: basic JS/Playwright examples

## How to run

1. Install dependencies (if there's a package.json):

```bash
npm install
```

2. Run tests or scripts using Playwright commands, e.g.:

```bash
npx playwright test
```
